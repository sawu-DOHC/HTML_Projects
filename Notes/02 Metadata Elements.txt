2. Metadata Elements (Within <head>):

<meta>: Provides metadata like character encoding, viewport settings, etc.

<title>: Defines the title of the webpage, displayed in the browser's tab.

<link>: Used to link external resources, such as stylesheets (CSS).

<style>: Defines CSS styles within the HTML document.

<script>: Links or contains JavaScript code within the page.

<meta>: Provides metadata such as keywords, descriptions, and author information.

<meta name="description" content="Description of the page"> helps with SEO.

<meta name="author" content="Author Name"> specifies the author of the page.
