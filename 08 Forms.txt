8. Forms:

<form>: Defines a form for user input.

<input>: Defines an input field (varies in type, such as text, password, checkbox).

<label>: Associates a label with a form element.

<select>: Creates a dropdown menu.

<option>: Defines options inside a <select> dropdown.

<textarea>: Defines a multi-line text input field.

<button>: Defines a clickable button.

<fieldset>: Groups related elements in a form and often includes a border.

<legend>: Provides a title or description for a <fieldset>.
